  Trackar is an application which purpose is to help organize work of car detailing studios.
</p>

<br/>
<br/>

## 🚀 Quick Start

**Installation**

```shell
  npm install
```

**Running development server**

```shell
  npm start
```

**Running tests**

```shell
  npm test
```

**Running storybook**

```shell
  npm run storybook
```

**Building project**

```shell
  npm run build
```

<br/>

## 📚 Technologies

- React
- Redux
- TypeScript
- SCSS modules
- Jest & testing-library
- date-fns
- Eslint & prettier
